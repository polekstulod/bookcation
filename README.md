
# Bookcation

Air bnb clone for listing reservations...


## Run Locally

Clone the project

```bash
  git clone https://github.com/polekstulod/bookcation
```

Go to the project directory

```bash
  cd  bookcation
```

Install dependencies

```bash
  npm install
```

Run Seeds
```bash
  node ./seeds/index
```

Start the server

```bash
  node index
```
